AI Multi-Agent System

The Master Key uses a modular AI architecture with four specialized agents.

---

1. Master Orchestrator
- Interprets tasks
- Normalizes context
- Routes tasks to other agents
- Provides high-level analysis

---

2. Repo Guardian
- Ensures repository hygiene
- Detects missing files or structural drift
- Recommends improvements

---

3. Governance Validator
- Enforces governance rules
- Validates permissions and workflows
- Flags compliance issues

---

4. Workflow Advisor
- Analyzes CI/CD pipelines
- Recommends optimizations
- Detects outdated or redundant jobs

---

Output Format

All agents return structured JSON for auditability.
